# 老板说分析一下

`boss-said-analyze-this` 是一个 Codex Skill，用来处理“老板让我分析一下”“这个你怎么看”“值不值得做”这类模糊分析需求。

它的核心不是套模板，而是把一句含糊的分析任务变成一个可交付判断：

1. 接收需求，必要时反问关键问题。
2. 补充并判断可分析物料。
3. 先确认分析方法或模型。
4. 先做表象分析，再做具体分析。
5. 输出“表象分析 / 具体分析 / 给领导可以这么说”。

## 适合场景

- 老板让分析一篇文章为什么火。
- 老板让分析一组业务数据为什么下滑。
- 老板让判断一个行业趋势值不值得做。
- 老板让分析一个产品、功能、竞品动作或方案机会。

## 仓库结构

```text
boss-said-analyze-this/
├── SKILL.md
└── references/
    ├── analysis-models.md
    └── scenario-playbooks.md
```

`agents/` 未包含在本仓库包中。核心 Skill 只依赖 `SKILL.md` 和 `references/`。

## 安装方式

把整个仓库目录放到 Codex 可读取的 skills 目录中，例如：

```bash
~/.codex/skills/boss-said-analyze-this/
```

或在你的工作区中作为本地 skill 使用。

## 示例调用

```text
老板让分析一下这篇文章为什么火。
```

```text
老板让分析一下最近 APP 新增为什么下降了。
```

```text
老板让分析一下目前 GEO 的趋势，值不值得做。
```

## 输出格式

默认输出三段：

```text
1. 表象分析
2. 具体分析
3. 给领导可以这么说
```
